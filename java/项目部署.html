# 远程机上部署项目

对这几天的远程部署做一下总结：

# 1、安装环境

# 1.1、安装nginx

  由于之前的项目已经安装了nginx，所以就只在nginx多加了一个配置文件，重启，需要注意的是，这里使用 ./nginx -s reload 完全不起作用，导致我后面折腾了半天。最后使用kill命令杀死进程，重新启动就ok

# 1.2、安装redis

虽然redis之前的项目也有安装过，但是对方要求需要新装一个，所以就老老实实的新装了一个，新装一个就要改端口，开放端口。然后带着配置文件启动就ok

oracle版本不一样。。。

解决办法：

-- 因为wm_concat在旧版本中不认识编译不了。需要加一个to_char
-- 修改前
select wm_concat(r.name)
-- 修改后
select to_char(wm_concat(r.name))
