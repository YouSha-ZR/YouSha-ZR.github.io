# 权限修饰符

java提供了四种访问权限修饰符，使用不同的访问权限修饰符是，被修饰的内容有不同的访问权限：

  • public：公共的所有地方都可以访问
  • protected 当前类，当前包，当前类的所有子类都可以访问
  • 缺省（没有修饰符） 当前包，当前类可以访问
  • private 私有的，当前类可以访问

public具有最大权限。private则是最小权限。

编写代码时，如果没有特殊的考虑，建议这样使用权限：

  • 成员变量使用private，隐藏细节;
  • 构造方法使用public，方便创建对象;
  • 成员方法使用public，方便调用方法;
  • 不加权限修饰符，就是default权限
